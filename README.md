# Deep-Learning-Practical-no-04
#  IMDB Movie Review Sentiment Analysis

##  Project Overview
This project implements Natural Language Processing (NLP) techniques to classify movie reviews as **Positive** or **Negative** using Machine Learning.

The model is trained on the IMDB dataset and uses preprocessing, TF-IDF vectorization, and a Naive Bayes classifier.

---

##  Objectives
- Apply NLP preprocessing techniques
- Convert text data into numerical form using TF-IDF
- Train a machine learning model for classification
- Evaluate model performance using metrics

---

##  Dataset
- Dataset: IMDB Movie Reviews
- Contains 50,000 reviews labeled as:
  - Positive
  - Negative
